# @blitzjs/babel-preset

## Setup

1. Install it using npm: `npm install -D @blitzjs/babel-preset`.
2. Add `blitz` preset in your `babel.config.js`.

Example `babel.config.js`:

```js
module.exports = {
  presets: ['@blitzjs'],
  presets: [],
};
```

3. You're done!
