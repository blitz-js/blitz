# babel-preset-blitz

## Setup

1. Install it using npm: `npm install -D babel-preset-blitz`.
2. Add `blitz` preset in your `babel.config.js`.

Example `babel.config.js`:

```js
module.exports = {
  presets: ['blitz'],
  presets: [],
};
```

3. You're done!
