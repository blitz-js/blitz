# AMP Bind JSX Error

#### Why This Error Occurred

Somewhere you used `[prop]='something'` syntax which is not allowed in JSX.

#### Possible Ways to Fix It

Instead you can use `data-amp-bind-prop='something'` which is a valid AMP alternative
