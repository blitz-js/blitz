# `Head` or `NextScript` attribute `crossOrigin` is deprecated

#### Why This Error Occurred

This option has been moved to `next.config.js`.

#### Possible Ways to Fix It

Add the config option:

```js
// next.config.js
module.exports = {
  crossOrigin: 'anonymous',
}
```
